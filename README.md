# SCM: Supply Chain Management Project

## Introduction
This is a Supply Chain Management Project for managing and optimize 

## Main Features
- Order management: entering orders, updating orders, tracking order status.
- Inventory management: monitoring stock levels, low‑stock alerts.
- Demand forecasting: forecasting module using statistical and machine learning models.
- Distribution optimization: distribution routing and allocation algorithms optimized for cost and time.
- Reporting and visualization: basic dashboard for KPIs.

## Technology Stack 
- Language: Python
- Web Framework: Django
- Database: PostgreSQL
- AI: pandas, numpy, scikit-learn, statsmodels, matplotlib, pytorch
